# Thread Craft

Write threads that people read to the end. The indie maker community popularized the Twitter thread as a growth format. Pieter Levels' "12 startups in 12 months" thread, IndieHackers revenue breakdowns, and MicroConf talk summaries are examples of threads that built audience and drove real business outcomes.

## Thread Openers

The first post determines whether anyone reads the rest. Proven opener formats:

| Format | Example | Why It Works |
|--------|---------|-------------|
| Bold claim | "Most productivity advice is wrong. Here's what actually works:" | Challenges beliefs, creates tension |
| Specific result | "I grew from 0 to 10K followers in 6 months. Here's exactly how:" | Concrete proof invites curiosity |
| Contrarian take | "Stop writing content. Start writing threads. Here's why:" | Pattern interrupt |
| Question | "What if everything you know about hiring is backwards?" | Engages the reader's own thinking |
| Story hook | "Last year I almost gave up on my startup. Then one conversation changed everything:" | Emotional investment |
| Listicle promise | "10 free tools that replaced $500/month of software:" | Clear value, scannable |

**Rules for openers:**
- Must stand alone as a compelling post even if nobody reads the thread
- No "Thread:" or "A thread" labels. They add nothing
- End with a colon or period that creates forward momentum
- Never start with "I" on the very first word. Rearrange the sentence

## Progressive Disclosure

Each post in a thread must earn the right to the next one. Indie creators who share their journey in threads use this technique naturally: each tweet reveals one step, one metric, one decision.

1. **Open a loop** in the first post (promise something specific)
2. **Deliver partial value** in each subsequent post (one idea, one step, one example)
3. **Close the loop** in the final post (deliver on the original promise)

**Structure options:**
- **Numbered list** -- "1/ Here's the first thing..." (most common, easiest to follow)
- **Story arc** -- Setup, rising tension, climax, lesson
- **Problem-solution pairs** -- Alternate between pain point and fix
- **Before/after** -- Show the wrong way, then the right way

## Cliffhanger Transitions

The last line of each post should pull the reader into the next one:

| Technique | Example |
|-----------|---------|
| Tease the next point | "But the third one changed everything." |
| Raise the stakes | "And that's when things got really interesting." |
| Ask a question | "So why doesn't anyone do this?" |
| Contrast | "That was the easy part. Here's where it gets hard:" |
| Incomplete thought | "The real reason is..." (continued in next post) |

## Thread Length Optimization

| Length | Best For | Read-Through Rate |
|--------|----------|------------------|
| 3-5 posts | Quick tip, single insight | 70-80% |
| 7-12 posts | Framework, step-by-step guide | 50-65% |
| 15-20 posts | Deep dive, comprehensive guide | 30-45% |
| 20+ posts | Mega-thread, course-style content | 20-30% |

**Sweet spot:** 7-12 posts. Long enough to demonstrate depth, short enough to maintain attention.

## Repurposing Threads

A single thread can become 5+ pieces of content, a key strategy from the indie maker playbook:

| Source | Becomes | How |
|--------|---------|-----|
| Full thread | Blog post | Expand each post into a paragraph, add headers |
| Full thread | Newsletter issue | Add personal commentary and a deeper example |
| Full thread | Carousel | One post per slide, add visuals |
| Full thread | Short-form video | Read the highlights over B-roll (60-90 seconds) |
| Individual post | Standalone tweet/post | Best-performing posts work as solo content |
| Thread opener | LinkedIn post | Expand the hook into a full-length LinkedIn post |

**Pro tip:** Write the thread first, repurpose second. Threads force concise thinking that makes every other format better.

## Numbering Conventions

| Platform | Convention | Example |
|----------|-----------|---------|
| Twitter/X | x/ format | "1/ Here's what I learned..." |
| LinkedIn | No numbers (use line breaks) | Each section is a paragraph, no thread markers |

## Key Rules

- Write the entire thread before posting the first post. Never thread in real-time
- Each post must deliver standalone value
- End the thread with a clear CTA: follow for more, bookmark this, share with someone who needs it
- Screenshot your best threads. They make great carousel content later
- Post threads between 8-10 AM in your audience's timezone for maximum reach
