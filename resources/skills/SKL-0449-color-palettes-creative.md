# Color Palettes: Creative & Agency

Palettes for portfolios, agencies, studios, and creative platforms. These palettes have more personality than SaaS/fintech but stay readable.

## Palette 1: Midnight Bold (dark creative)

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `#E11D48` | Brand accent, CTAs, highlights |
| Secondary | `#8B5CF6` | Secondary actions, tags |
| Background | `#0A0A0A` | Dark canvas (work pops against black) |
| Surface | `#171717` | Cards, panels |
| Text | `#FAFAFA` | Body text (18.1:1 on black) |
| Text Secondary | `#A3A3A3` | Captions, metadata (9.2:1 on black) |
| Border | `#262626` | Subtle dividers |

**Light mode:** Background `#FAFAFA`, Surface `#FFFFFF`, Text `#0A0A0A`

| Role | Hex | Usage |

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `#06B6D4` | Brand, interactive elements |
| Accent | `#F43F5E` | Highlights, notifications |
| Background | `#FAFAFA` | Clean white |
| Surface | `#FFFFFF` | Cards |
| Text | `#09090B` | Body text (17.4:1 on white) |
| Text Secondary | `#52525B` | Supporting text (6.9:1 on white) |

**Dark mode:** Background `#09090B`, Surface `#18181B`, Text `#FAFAFA`

## Creative-Specific Rules

- **Dark backgrounds make visual work pop.** Default to dark for portfolios showcasing photography, design, or video.
- **One bold accent is enough.** Creative doesn't mean chaotic. One vibrant color against neutrals is more striking than a rainbow.
- **Typography carries more weight than color** in creative sites. Invest in a distinctive display font.
- **Whitespace is a design element.** More breathing room = more premium feel. Creative sites should be generous with spacing.
- **Break one convention, not all of them.** An unconventional layout + conventional navigation = creative but usable. Unconventional everything = frustrating.
