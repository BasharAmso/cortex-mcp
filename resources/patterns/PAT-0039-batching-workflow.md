# Content Batching Workflow

Separate thinking from doing. Batch by task type, not by post.

## The Content Production Pipeline

```
Ideate (30 min) -> Write (2 hrs) -> Edit (1 hr) -> Design (1 hr) -> Schedule (30 min)
```

The key insight: do each stage for ALL posts before moving to the next. Writing 5 posts in a row is faster than writing, editing, designing, and scheduling one post five times.

## Batch Day vs. Daily Creation

| Approach | Best For | Schedule |
|----------|----------|----------|
| **Weekly batch** | 3-5 posts/week, one platform | One 3-4 hour block per week |
| **Biweekly batch** | 5-7 posts/week, multi-platform | Two 2-hour blocks per week |
| **Monthly batch** | Planned campaigns, evergreen content | One full day per month + weekly touch-ups |
| **Daily creation** | Reactive/news content, personal journaling | 30-45 min daily (not recommended as sole approach) |

## Time-Blocking Template (Weekly Batch)

| Block | Duration | Task |
|-------|----------|------|
| **Block 1: Ideate** | 30 min | Review idea bank, pick 5 topics, write one-line hooks |
| **Block 2: Draft** | 90 min | Write all 5 drafts back-to-back, no editing |
| **Block 3: Edit** | 45 min | Polish all drafts, tighten hooks, check CTAs |
| **Block 4: Design** | 45 min | Create all visuals, carousels, or thumbnails |
| **Block 5: Schedule** | 15 min | Load everything into scheduler with publish times |

Total: ~3.5 hours for a full week of content.

## Batching Rules

1. **Never edit while writing.** Drafting and editing use different brain modes. Switching kills speed.
2. **Use templates.** Have 3-5 post formats you rotate through (story, list, question, how-to, opinion). Start from structure, not blank page.
3. **Set a timer per draft.** 15-20 minutes max. If it's not flowing, move to the next and come back.
4. **Batch design separately.** Open Canva/Figma once, make all graphics in one session.
5. **Schedule with buffer.** Always have 2-3 posts in reserve so one bad week doesn't break the streak.

## Recommended Tool Stack

| Stage | Tools |
|-------|-------|
| Ideate | Notion, Apple Notes, voice memos |
| Write | Google Docs, Notion, drafts app |
| Edit | Hemingway App, read aloud, peer review |
| Design | Canva, Figma, CapCut (video) |
| Schedule | Buffer, Later, Typefully, native schedulers |

## Post Format Templates

- **Story:** Hook (1 line) -> Context (2 lines) -> Turning point -> Lesson -> CTA
- **List:** Bold claim -> 5-7 items (one line each) -> Closing insight
- **How-To:** Problem (1 line) -> Steps (numbered) -> Result -> "Try this"
- **Hot Take:** Contrarian opener -> Why most people get it wrong -> Your take -> Question

## Anti-Patterns

- Batching without an idea bank (you'll waste 60% of the session staring at a blank screen)
- Perfectionism during drafting (write ugly first drafts, fix later)
- Skipping the schedule step (batched content in a folder is not content that gets posted)
