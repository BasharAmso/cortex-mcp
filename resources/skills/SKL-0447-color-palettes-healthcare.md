# Color Palettes: Healthcare & Wellness

Palettes for patient portals, telehealth, wellness apps, and medical tools. Healthcare demands calm, trust, and exceptional readability.

## Palette 1: Clinical Blue (most common in health)

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `#0284C7` | Navigation, headers, trust elements |
| Primary Light | `#38BDF8` | Links, interactive elements |
| Accent | `#14B8A6` | Wellness indicators, positive health |
| Background | `#F0F9FF` | Soft blue-tinted background |
| Surface | `#FFFFFF` | Cards, forms, panels |
| Text | `#0C4A6E` | Body text (7.8:1 on white) |
| Text Secondary | `#64748B` | Labels, metadata (4.6:1 on white) |
| Success | `#16A34A` | Normal results, healthy ranges |
| Warning | `#D97706` | Borderline results, attention needed |
| Critical | `#DC2626` | Abnormal results, urgent alerts |

**Dark mode:** Background `#0C1929`, Surface `#1A2D42`, Text `#E0F2FE`

## Healthcare-Specific Rules

- **Readability is non-negotiable.** Users may be elderly, stressed, or in bright clinical lighting. Minimum 5:1 contrast for all text.
- **Avoid pure red for non-critical info.** Red triggers anxiety in medical contexts. Reserve it for genuinely urgent/abnormal results.
- **Blue = trust** in healthcare globally. When in doubt, go blue.
- **Never use color alone** to convey health status. Always pair with text labels, icons, or patterns.
