# Carousel Design

Create carousel posts that get saved, shared, and establish authority. Carousels are the highest-reach format on LinkedIn and one of the most saved formats on Instagram. Indie makers and bootstrapped founders use carousels to package frameworks, tool lists, and step-by-step processes into shareable assets that compound audience growth over time.

## Slide Structure: Hook to CTA

Every carousel follows the same skeleton:

| Slide | Purpose | Content |
|-------|---------|---------|
| 1 (Cover) | Stop the scroll | Bold claim, question, or promise. Treat it like a thumbnail |
| 2 | Context | Why this matters. Set up the problem or frame the topic |
| 3-8 | Value slides | One idea per slide. Numbered or titled for scannability |
| 9 | Summary | Recap key points in a single visual (optional but powerful) |
| 10 (CTA) | Action | Follow, save, share, comment, or visit link |

**Cover slide rules:**
- Maximum 8 words. If you need more, the hook isn't sharp enough
- Use a contrasting background color or bold typography
- Include your name/handle small in the corner for attribution when shared
- Test: would someone screenshot just this slide? If not, rewrite it

## Text-to-Visual Ratio

| Content Type | Text | Visual |
|-------------|------|--------|
| Educational (tips, frameworks) | 70% text, 30% visual | Icons, simple diagrams |
| Storytelling (journey, case study) | 50% text, 50% visual | Photos, illustrations |
| Data-driven (stats, research) | 40% text, 60% visual | Charts, graphs, callout numbers |
| Inspirational (quotes, mindset) | 30% text, 70% visual | Strong imagery, large typography |

**General rule:** If a slide has more than 40 words, split it into two slides. Mobile readers scroll fast.

## Platform Sizing

| Platform | Dimensions | Aspect Ratio | Format |
|----------|-----------|--------------|--------|
| LinkedIn | 1080 x 1350 px | 4:5 | PDF upload (not images) |
| Instagram | 1080 x 1350 px | 4:5 | Image carousel (up to 20 slides) |
| Twitter/X | 1600 x 900 px | 16:9 | Image carousel (up to 4 images) |

**LinkedIn-specific:** Upload as PDF for swipe behavior. Image carousels on LinkedIn do not get the same swipe UX and perform worse. Name the PDF file something descriptive since it shows as the document title.

## Design Templates for Indie Makers

These proven layouts work well for bootstrapped founders sharing knowledge:

1. **The Listicle** -- "7 tools I use daily" -- numbered slides, consistent layout, icon per item
2. **The Framework** -- "My 4-step process for X" -- diagram on slide 2, one step per slide after
3. **The Build-in-Public Update** -- Revenue screenshot, lesson learned, next milestone
4. **The Myth Buster** -- "5 things you think are true but aren't" -- myth vs. reality per slide
5. **The Before/After** -- "What I did wrong vs. what works" -- split-screen comparison slides
6. **The Case Study** -- "How I went from X to Y" -- timeline progression with key moments

## Canva Workflow

1. Start from a blank presentation (1080x1350 custom size)
2. Design slide 1 (cover) and slide 10 (CTA) first. These bookend the experience
3. Create one "value slide" template with your fonts, colors, and layout
4. Duplicate the template for slides 3-8. Change only the content
5. Export as PDF for LinkedIn, as PNG for Instagram
6. Save your template as a Canva brand template for reuse

**Typography:** Use 2 fonts maximum. One for headlines (bold/heavy), one for body. Minimum 24pt for body text on mobile.

**Colors:** Pick 2-3 brand colors. Use one as the primary background, one for accents, one for text. Consistent color builds recognition across posts.

## What Makes Content Save-Worthy

People save content they want to reference later:

- Checklists and step-by-step processes
- Tool recommendations with specific use cases
- Frameworks they can apply immediately
- Data and statistics they want to cite
- Templates and swipe files

**Test:** Before publishing, ask "Would I bookmark this to come back to?" If no, add more specificity or a concrete example.

## Key Rules

- One idea per slide. If you're cramming, you need more slides
- Design for mobile first. Most social media is consumed on phones
- Consistent branding across carousels builds recognition over time
- The cover slide is 80% of your carousel's performance. Spend the most time there
- Always include a CTA on the final slide. Never end abruptly
