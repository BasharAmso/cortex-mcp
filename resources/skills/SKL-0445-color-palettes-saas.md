# Color Palettes: SaaS & B2B

Ready-made palettes for dashboards, admin panels, and productivity tools. Each palette includes WCAG AA-compliant text pairings.

## Palette 1: Trust Blue (most common SaaS choice)

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `#2563EB` | CTAs, active states, links |
| Primary Dark | `#1D4ED8` | Hover states, selected items |
| Accent | `#F59E0B` | Alerts, badges, highlights |
| Background | `#F8FAFC` | Page background |
| Surface | `#FFFFFF` | Cards, panels |
| Text | `#0F172A` | Body text (13.8:1 on white) |
| Text Secondary | `#64748B` | Labels, captions (4.6:1 on white) |
| Border | `#E2E8F0` | Dividers, card borders |
| Success | `#16A34A` | Positive states |
| Error | `#DC2626` | Errors, destructive actions |

**Dark mode:** Background `#0F172A`, Surface `#1E293B`, Text `#F1F5F9`, Border `#334155`

## When to Pick Which

| Palette | Best For |
|---------|----------|
| Trust Blue | General SaaS, CRM, project management, analytics |
| Indigo Calm | Creative tools, design platforms, developer tools |
| Emerald Focus | Finance-adjacent, health metrics, performance dashboards |
